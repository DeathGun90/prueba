¿Que son los sistemas operativos?

Los sitemas operativos son parte fundamental del software de un computador, tablet o smartphone, es por ello que al momento de utilizar cualquiera de estos dispositivos se vuelve fundamental conocer acerca del tema, para con ello saber de algunas ventajas y desventajas que puede tener el sistema operativo que estemos utilizando.

Existen 2 tipos de sistemas operativos:

  • Privativos: Se le conoce como software privativo al cual no tiene forma de acceso existente a su codigo fuente
  • Libres: Son aquellos que permiten que las personas puedan alterar el codigo fuente sin restricciones, con ellos se podria decir que el usuario puede usar el programa con cualquier proposito

    • ¿Que es Linux?

    Linux es la denominación generalizada que reciben una serie de sistemas operativos de tipo Unix, los cuales son tambien de codigo abierto, multiplataforma, etc. El hecho de que Linux sea de codigo abierto permite a las empresas, instituciones o usuarios alterar de manera libre este sistema operativo.

    Linux tiene diferentes distribuciones, algunas de ellas son:

  • Debian
  • Ubuntu
  • Fedora
  • OpenSUSE
  • Arch Linux
  • Manjaro

    • ¿Cuando y Quien creo Linux?

    Linux comenzo siendo un proyecto personal del estudiante de informatica de la Universidad de Helsinki, Finlandia, Linus Torvalds, que tenia como proposito crear un nuevo nucleo de sistema operativo libre, este fue lanzado en el año 1991. Su objetivo fundamental era crear un núcleo del S.O. que fuera totalmente compatible con el de UNIX y que permitiera ejecutar todos los programas gratuitos compatibles UNIX desarrollados por la Free Software Foundation.
